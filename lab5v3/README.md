# Контрольні запитання

1.Що таке generics? Які їхні переваги?
Generics — узагальнені типи/методи.
Переваги: типобезпечність, повторне використання коду, без кастингу, продуктивність.

2.Які основні відмінності між ArrayList і List<T>?
ArrayList vs List<T>
ArrayList — будь-які об’єкти, потрібен кастинг.
List<T> — конкретний тип, типобезпечний, швидший.

3.Чим відрізняється Dictionary<TKey,TValue> від List<T>?
Dictionary<TKey,TValue> vs List<T>
List<T> — доступ за індексом.
Dictionary — доступ по ключу (ключ-значення).

4.У чому перевага LINQ над класичними циклами?
LINQ vs цикли
Лаконічно, декларативно, читається легше, працює з різними джерелами даних.

5.Як працює ключове слово finally?
Виконується завжди, навіть при винятку.
Використовується для очищення ресурсів.

6.Коли доцільно створювати власні класи винятків?
Коли стандартні не підходять, для чіткої обробки помилок або бізнес-логіки.

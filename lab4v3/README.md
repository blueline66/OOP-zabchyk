# Лабораторна робота №4
## Тема
Абстракції та інтерфейси. Композиція та агрегація.

## Мета
Навчитися створювати абстрактні класи та інтерфейси, будувати ієрархії класів із використанням композиції та агрегації, реалізовувати обчислення, демонструвати гнучкість і повторне використання коду.

## Опис роботи
Було створено інтерфейс `ITextFilter` і абстрактний клас `AbstractFilter`.  
Реалізовано два фільтри:
- `SpaceRemoverFilter` — видаляє пробіли з рядка.
- `BannedWordsFilter` — замінює заборонені слова зірочками.

Використано **композицію** у класі `TextProcessor`, який зберігає список фільтрів і послідовно застосовує їх до тексту.

# Lab 3 (Наслідування) — Варіант: Тварини

## Тема та мета
Тема: Наслідування: основи.  
Мета: закріпити знання про базові та похідні класи, використання base(...), virtual/override та демонстрацію поліморфізму на колекції об'єктів.

## Опис виконання
- Створено базовий абстрактний клас Animal з методами Speak() і Eat().
- Реалізовано два похідних класи: Dog і Cat.
- Конструктори похідних класів викликають base(...).
- Демонстрація поліморфізму через List<Animal>.
